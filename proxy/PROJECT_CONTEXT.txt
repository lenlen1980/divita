# DIVITA Proxy - Контекст проекта (для Warp UI)

## Основная информация:
- **Сервер**: Ubuntu, Nginx
- **Каталог**: /var/www/html/proxy
- **Ресурсы**: локализованы и проксируются

## Основные задачи:
- Обход блокировок в РФ
- Поддержка стабильного доступа

## Ключевые скрипты:
- `deploy.sh`: Обновление конфигурации
- `monitor.sh`: Мониторинг состояния
- `test-proxy.sh`: Тестирование эндпоинтов

## Рекомендуемые действия:
1. Выполнить `monitor.sh` для проверки состояния
2. Запустить `test-proxy.sh` для тестирования доступности
3. Обновлять конфигурацию через `deploy.sh` после изменений
